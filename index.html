jQuery CONTEXTMENU v1.1.0

contextmenu is a context menu (right click menu) jQuery plugin.

I developed it to use on my Uygulama project's new version.

I've only tested it on

  • Chromium (14.0.835.202),
  • Google Chrome (15.0.874.120),
  • Mozilla Firefox (3.6.17),
  • Opera (11.52)
  • Internet Explorer (8.0.6001)

and it works enough for me for now!

USAGE

<script src="contextmenu.js"></script>
<link rel="stylesheet" href="contextmenu.css" />
<script>
  $(document).contextmenu({
    menu: {
      'custom link' : 'http://...'
    }
  });
</script>

OPTIONS

                    init   : Optional init function.
                    start  : Optional start function
                    finish : Optional finish function.
                    ctrl   : Contect menu disable. If you right click with ctrl key 
                             the contect menu do not show. [true,false] Default value is true;
                    style  : Optional class name.
                    menu   : Required menu object.
